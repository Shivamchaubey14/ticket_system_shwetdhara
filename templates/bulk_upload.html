Employee Upload

Upload the employee master sheet — create or update staff records. Unique key: Employee Code. Manager links resolved in a second pass.

Row-validated Upsert on Emp. Code Manager links auto-resolved
Required & Supported Columns First row = header  ·  Aliases resolved automatically

Required

Employee Code / Emp Code First Name / Name

Optional columns (alias-resolved)

Last Name / Surname Department / Dept Designation / Title / Role Grade / Employee Grade Branch / Work Location Employee Email Id / Email Employee Phone Number / Mobile Reporting Manager / Manager Code Functional Manager / Func. Mgr Code
Import runs in two passes: employees are created/updated first, then Reporting Manager codes are resolved to FK links in a second sweep — so a manager imported in the same file will be correctly linked. If an email is blank or missing, a placeholder <employee_code>@shwetdhara.local is generated automatically. Passwords are set unusable; employees must use password-reset to log in.
